Install Resend

02

Add Environment Variable

In Vercel dashboard → Settings → Environment Variables, add:

.env.local
env 
RESEND_API_KEY=re_xxxxxxxxxxxxxxxx

Note: Get your API key from resend.com → API Keys. The key stays server-side only — it is never exposed to the browser.

03

Create the API Route

This single route sends both emails in one Resend batch call.

app/api/contact/route.js
javascript 
import { Resend } from 'resend';

const resend = new Resend(process.env.RESEND_API_KEY);

export async function POST(req) {
  const { name, email, phone, message } = await req.json();

  const { error } = await resend.batch.send([

    // Email 1: Notify your team
    {
      from: 'website@gruha.ai',
      to: 'team@gruha.ai',
      replyTo: email,         // reply goes directly to the lead
      subject: `New enquiry from ${name}`,
      html: `
        <h2>New enquiry — Gruha.ai</h2>
        <p><strong>Name:</strong> ${name}</p>
        <p><strong>Email:</strong> ${email}</p>
        <p><strong>Phone:</strong> ${phone}</p>
        <p><strong>Message:</strong> ${message}</p>
      `,
    },

    // Email 2: Confirmation to the lead
    {
      from: 'website@gruha.ai',
      to: email,
      replyTo: 'team@gruha.ai',  // lead's reply lands in your inbox
      subject: "We've received your enquiry",
      html: `
        <p>Hi ${name},</p>
        <p>
          Thank you for reaching out to Gruha.ai. We've received your
          enquiry and will get back to you within 1 business day.
        </p>
        <p>Here's a copy of what you sent us:</p>
        <blockquote style="border-left:3px solid #ccc; padding-left:12px; color:#555">
          ${message}
        </blockquote>
        <p>Feel free to reply to this email if you have anything to add.</p>
        <p>— Team Gruha.ai</p>
      `,
    },
  ]);

  if (error) return Response.json({ error }, { status: 500 });
  return Response.json({ success: true });
}
04

The Contact Form (React)

components/ContactForm.jsx
jsx 
'use client';
import { useState } from 'react';

export default function ContactForm() {
  const [status, setStatus] = useState('idle'); // idle | sending | done | error

  async function handleSubmit(e) {
    e.preventDefault();
    setStatus('sending');

    const data = Object.fromEntries(new FormData(e.target));

    const res = await fetch('/api/contact', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(data),
    });

    setStatus(res.ok ? 'done' : 'error');
  }

  return (
    <form onSubmit={handleSubmit}>
      <input name="name"    type="text"  placeholder="Name"    required />
      <input name="email"   type="email" placeholder="Email"   required />
      <input name="phone"   type="tel"   placeholder="Phone"   />
      <textarea name="message"           placeholder="Message" />

      <button type="submit" disabled={status === 'sending'}>
        {status === 'sending' ? 'Sending...' : 'Submit'}
      </button>

      {status === 'done'  && <p>Thanks! We'll be in touch.</p>}
      {status === 'error' && <p>Something went wrong. Please try again.</p>}
    </form>
  );
}
05

Verify Your Domain on Resend

In Resend dashboard → Domains → Add Domain → add gruha.ai and apply the DNS records they provide. This allows sending from a @gruha.ai address.

Tip: DNS propagation usually takes ~10 minutes. Until verified, emails will either fail or land in spam.
